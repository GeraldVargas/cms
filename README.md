#cms(noti)
Trabajo práctico para la materia de Generación de Software.

## Enlaces del Proyecto

El proyecto se encuentra desplegado y funcionando en los siguientes enlaces:

* **Vista Pública,Sitio web:** [https://cms-eta-sable.vercel.app/](https://cms-eta-sable.vercel.app/)
* **Panel de Administración:** [https://cms-eta-sable.vercel.app/admin](https://cms-eta-sable.vercel.app/admin)

> **Nota sobre el acceso:** Para esta entrega, el panel de administración **no requiere inicio de sesión (login)**. Puedes acceder directamente al panel de control y gestionar el contenido utilizando el enlace de administrador proporcionado arriba (agregando `/admin` a la URL principal).

## Estructura del Repositorio

* `frong/`: Contiene todo el código del Frontend.
* `backg/`: Contiene el código del Backend.
